• Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.

  • Cascading Style Sheets is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

  • JavaScript is a high-level, often just-in-time compiled, programming language that conforms to the [Link: ECMAScript specification https://developer.mozilla.org/en-US/docs/Web/JavaScript], enabling interactive web page content and functionality. It's a core technology of the World Wide Web, allowing for dynamic updates, animations, and other interactive elements on websites.

  • React is a JavaScript library for building user interfaces, specifically focusing on creating interactive and dynamic UIs for web applications. It's component-based, meaning that UIs are built from reusable and self-contained pieces of code called components.

  • Node.js is an open-source, cross-platform JavaScript runtime environment that allows developers to execute JavaScript code outside of a web browser. It runs on the V8 JavaScript engine, which is the same engine that powers Google Chrome.
